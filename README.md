# Face Recognition Attendance System

An AI-based attendance system that automatically marks attendance using face recognition.

## Technologies Used
- Python
- OpenCV
- Raspberry Pi
- Webcam

## Features
- Detects faces using computer vision
- Recognizes registered users
- Automatically records attendance
