# Face Recognition Attendance System

An AI-powered attendance system that automatically records attendance using real-time face recognition.  
The system detects and recognizes faces from a webcam feed and stores attendance records in a CSV file.

This project demonstrates the use of computer vision and deep learning for automating attendance systems.

---

## Technologies Used

- Python
- OpenCV
- PyTorch
- FaceNet
- MTCNN
- NumPy
- Raspberry Pi / Webcam

---

## Features

- Real-time face detection using MTCNN
- Face recognition using FaceNet deep learning model
- Automatic attendance logging
- Stores attendance data in CSV format
- Face embedding caching for faster recognition
- Works with webcam input
